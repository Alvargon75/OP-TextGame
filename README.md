# Juego de One Piece

## Intro

Esto es un simple juego que estoy haciendo como práctica de JavaScript.

## ¿Cómo Jugar?

1. Descargar como archivo comprimido (ZIP).
2. Extrae el juego.
3. Abre `index.html` en tu navegador favotiro y juega.
